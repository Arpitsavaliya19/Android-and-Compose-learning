# Android-and-Compose-learning
Learning repository for Android Jetpack Compose with multiple mini-projects covering MVVM, navigation, UI, and animations.
ComposeLearningRepo

ComposeLearningRepo is a personal Android development learning repository focused on Jetpack Compose, MVVM architecture, navigation, and other modern Android concepts. This repository contains multiple small projects, each designed to explore and practice a specific topic in Compose and Android development.

Purpose

Experiment with and understand Jetpack Compose components, layouts, and theming.

Implement MVVM architecture with ViewModels, repositories, and data models.

Practice navigation between screens using Compose Navigation.

Develop hands-on experience with Android app development through mini-projects.

Projects Included

Project1_CupcakeApp – Practice state management, UI design, and navigation using a sample cupcake ordering app.

Project2_NavigationDemo – Focused on Compose navigation patterns and passing arguments between screens.

Project3_Animations – Practice animations, transitions, and gestures in Compose.

Project4_… – (Add other projects as you create them, each covering a specific learning topic.)

How to Use

Each project is self-contained and can be opened in Android Studio individually.

Use the README.md inside each project folder for details on its functionality and topics covered.

Experiments and temporary files are kept separate per project to avoid conflicts.

Learning Goals

Master Jetpack Compose basics and advanced UI techniques.

Implement clean architecture and MVVM patterns in small projects.

Gain confidence in navigation, state handling, and testing in Android Compose.

Build a portfolio of mini-projects to demonstrate Compose skills.
